## 📘 Description
This project presents the design and deployment of a professional IT and network infrastructure for **AtlasCom**, a mid-sized service company with around 100 employees.  
The objective is to provide a **secure, scalable, and efficient digital environment** supporting daily operations and inter-departmental communication.

---

## 🎯 Objectives
- Design a structured local network with **VLAN segmentation**  
- Deploy essential services: **DHCP, DNS, Active Directory, VPN**  
- Ensure network **security and monitoring**  
- Provide **documentation** and configuration files for maintenance  
- Enable **scalability** for future expansions

---

## ⚙️ Technologies & Tools
| Category | Tools |
|----------|-------|
| Network | Cisco Packet Tracer, GNS3 |

| Virtualization | VMware, VirtualBox |
| Scripting & Automation | Python, PowerShell |

---

## 🖥️ Network Architecture
- **Router Cisco ISR** – Main gateway  
- **L3 Switches** – Inter-VLAN routing  
- **L2 Switches** – Department distribution  
- **Windows Server** – Active Directory, DHCP, DNS  
- **VPN Server** – Secure remote access  
- **Firewall** – pfSense or Fortigate  
- **Wi-Fi Access Points** – Internal wireless connectivity

## 🔐 Security Recommendations
- VLAN isolation between departments  
- Strong password policies and regular updates  
- VPN for remote connections  
- Regular backups and patch management

## 🧩 Implementation Steps
1. Requirement analysis & network planning  
2. Physical and logical topology design  
3. Hardware installation and configuration  
4. VLANs, routing, DHCP, DNS setup  
5. Security implementation (firewall, VPN)  
6. Monitoring & testing  
7. Documentation and final report
